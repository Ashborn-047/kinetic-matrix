**Kinetic Matrix** is a premium, cyberpunk-inspired web experience and interactive component library. It explores the synergy between brutalist digital aesthetics and fluid motion design, featuring matrix-rain environments, kinetic typography, and high-performance React experiments.

![Kinetic Matrix Preview](https://raw.githubusercontent.com/Ashborn-047/kinetic-matrix/main/public/preview-hero.png) *(Note: Add a real preview image to your public folder)*

## âš¡ Features

- **Matrix Rain Environment**: High-performance character rain with customizable density and speed.
- **Kinetic Typography**: Scrambled text effects and morphing weights that react to user interaction.
- **Brutalist UI**: A high-contrast "system-locked" aesthetic using green-on-black palettes.
- **Interactive Experiments**: A dashboard of experimental UI components (Neural Scraper, Matrix Contrast, etc.).
- **GSAP Driven Motion**: Smooth, frame-perfect animations and scroll-triggered transformations.
- **Modern Tech Stack**: Built with React 18, Vite 6, and Tailwind CSS 4.

## ğŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ashborn-047/kinetic-matrix.git
   cd kinetic-matrix
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
